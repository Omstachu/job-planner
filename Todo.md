# Authentication

    * Sign up
    * Login
    * Logout
    * Admin Account

# Admin

    * Dashboard
        - See available jobs
        - See Completed Jobs
        - Change status of jobs. Only admin can approve the completion of a job
    * Post Jobs
    * Delete Jobs

# User

    * Dashboard
        - See available Jobs
        - See completed Jobs
        - Accept available jobs
            - Accepted jobs cant overlap in scheduled time
        - Change status of job. (Accepted, In progress, Completed, Cancelled)

# Search/Filter

    * Text
    * Start and End date
    * Location
